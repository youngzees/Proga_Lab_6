ВЫВОД

В результате данной работы были выполнены четыре задания: 

обработка матрицы с поиском отрицательных элементов;
шифрование AES-128 OFB с выводом всех ключей и промежуточных состояний;
решение СЛАУ методом LU-разложения и Зейделя с точностью 0.001;
поиск минимального числа ходов для приведения доски к шахматному паттерну алгоритмом BFS. 
Все программы реализованы на C++ или Go, работа выполнена в полном объёме, все программы работают стабильно и дают корректные результаты.
